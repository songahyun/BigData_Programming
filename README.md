# BigDataPrigramming: 지역별 인프라 변수 생성을 통한 상권분석
※ This repository reorganizes my individual contributions from a team project conducted as part of a university course.  
※ 이 repository는 대학교 수업의 일환으로 진행된 팀 프로젝트에서 개인적으로 기여한 부분을 재정리한 것입니다.

---
### 프로젝트 배경:
자영업 창업 시 상권의 특성과 유동 인구, 업종 분포 등의 요인을 충분히 고려하지 않으면 매출 부진이나 폐업으로 이어질 가능성이 높다. 
그러나 예비 창업자가 다양한 상권 데이터를 종합적으로 분석하여 적합한 입지를 선정하는 것은 쉽지 않으며, 객관적인 데이터 기반 상권 분석의 필요성이 증가하고 있다. 따라서 상권 관련 데이터를 활용하여 지역별 상권 특성을 분석하고, 창업 의사결정을 지원할 수 있는 분석 방법이 요구된다.

### 프로젝트 목표:
- 상권 데이터를 활용한 지역별 상권 특성 분석
- 유동 인구 및 업종 분포 기반 상권 비교 분석
- 데이터 기반 상권 분석 방법 제시
- 창업 입지 선정 의사결정 지원
- 상권 정보에 대한 접근성 향상

### 사용 데이터:
- 서울통계통합플랫폼의 행정구역(구별)
- 서울 열린데이터 광장의 서울시 등록인구(연령별/동별) 통계
- 서울 열린데이터 광장의 서울시 문화공간(전시시설) 통계
- 서울 열린데이터 광장의 서울시 연도별 정류장 현황
- 서울 열린데이터 광장의 서울시 의료기관(구별)(20210년 이후) 통계
- 서울 열린데이터 광장의 서울시 상권분석서비스(상권변화지표-자치구)
- 서울 열린데이터 광장의 서울시 학교 기본정보
- 공공 데이터 포털: 서울교통공사_자치구별지하철역정보
- KOSIS 국가통계포털의 시군구별 동별 요양기관 현황
